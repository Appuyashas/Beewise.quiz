📋 How to Create a CSV File

Your CSV needs exactly these 5 column headers. opt0 is always the correct answer — the app shuffles the order automatically so students won't know which one it is.

questionopt0 (correct)opt1opt2opt3
What is the capital of INDIA? New DelhiMumbaiChenniKolkata
Which planet is closest to the Sun? MercuryVenusEarthMars

You can also add an optional category column to label your questions.

question,opt0,opt1,opt2,opt3,category What is the capital of France?,Paris,London,Berlin,Rome,Geography Which planet is closest to the Sun?,Mercury,Venus,Earth,Mars,Science Who wrote Romeo and Juliet?,William Shakespeare,Charles Dickens,Jane Austen,Leo Tolstoy,Literature
💡

Easiest way to make one: Open Microsoft Excel or Google Sheets. Put questions in column A, correct answer in B, wrong options in C, D, E. Then go to File → Save As → CSV. Upload it here!

🤖

Let AI make your CSV in seconds! Copy this prompt and paste it into Claude, ChatGPT, or any AI:

Generate 10 quiz questions about [YOUR TOPIC] in CSV format with exactly these columns: question,opt0,opt1,opt2,opt3,category Rules: - opt0 must ALWAYS be the correct answer - opt1, opt2, opt3 are wrong answers - No header row, just the data rows - No quotes around values unless the text contains a comma Example row: What is the capital of France?,Paris,London,Berlin,Rome,Geography

👆 Click the box to select all · Replace [YOUR TOPIC] with e.g. "NEET Biology", "World History", "Python basics"

📱 On Phone:
1️⃣ Open Claude.ai or ChatGPT app
2️⃣ Paste the prompt above and send it
3️⃣ Press and hold the AI's reply → Select All → Copy
4️⃣ Open Google Sheets app → paste into cell A1
5️⃣ Tap File → Download → CSV — save to your phone
6️⃣ Come back here and upload that file 🎉
💻 On PC:
1️⃣ Copy the prompt → paste into Claude.ai or ChatGPT
2️⃣ Copy the AI's output → paste into Notepad
3️⃣ Save as questions.csv (change "Save as type" to All Files)
4️⃣ Upload it above ✅
